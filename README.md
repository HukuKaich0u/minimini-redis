# minimini-redis

[日本語 / Japanese](ja.md)

### Purpose

This repository is a personal study project for learning by reimplementing `mini-redis`, the sample project from `tokio`.

The goal is not to maintain a verbatim copy of the original implementation, but to understand `tokio`, async programming, and design decisions by building it myself.

References:

- Tokio: <https://tokio.rs/>
- mini-redis: <https://github.com/tokio-rs/mini-redis>
